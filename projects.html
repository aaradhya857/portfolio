Hand Gesture-Based Volume Control

. Volume Control Using Hand Gestures

computer vision project that allows users to control system volume using hand gestures detected through a webcam. The project tracks the distance between specific fingers (thumb and index) to increase or decrease volume, providing a touchless user interface.

Role: Solo developer

Technologies: Python, OpenCV, MediaPipe, pycaw (Windows Audio Control)

Key Features:

Real-time hand tracking

Smooth gesture-to-volume mapping

Visual feedback with on-screen volume bar

ESPWizard – Smart Remote

ESPWizard – Smart ESP32 Remote App

A React Native-based mobile application that acts as a universal remote for ESP32-powered devices. The app dynamically adapts its interface depending on the selected device type, ensuring a tailored user experience. Additionally, it streams live camera feed from the bot to the app, enabling real-time monitoring and control.

Role: Lead developer for UI and communication protocols

Technologies: React Native, JavaScript, TCP/IP, UDP

Key Features:

Auto-customized UI for each device

Real-time command execution

Live camera streaming

Outcome: Final-year robotics project praised for its flexibility and seamless hardware-software integration.
